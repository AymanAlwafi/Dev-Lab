Saudi Real Estate AI

Machine learning project that classifies Saudi property listings as "Affordable" or "High-Value" using Random Forest — built as a Data Mining course final project.

Python Scikit-learn Random Forest Pandas Data Mining

About

This was the final project for CIS 517 — Data Mining, at Imam Abdulrahman bin Faisal University. The goal was to apply the full data mining pipeline on a real-world dataset: cleaning, preprocessing, building models, tuning them, and interpreting the results.

We worked with a dataset of 5,000+ Saudi real estate listings from Kaggle, which came with challenges: Arabic text encoding, missing values, extreme outliers from commercial properties, and heavily skewed price distributions. The end result was a Random Forest classifier that achieved 87.6% accuracy and an AUC score of 0.95.

Methodology

  • Data Cleaning — Handled UTF-8-SIG Arabic encoding, dropped irrelevant columns (IDs, URLs, images), and encoded city names as integers for model compatibility.
  • Outlier Removal — Applied the IQR method on property area to remove extreme values (commercial buildings, palaces) that skewed the distribution. Dataset reduced from ~5,200 to ~4,850 usable records.
  • Target Engineering — Converted continuous price into a binary classification target: 0 (Affordable) for listings at or below the median, 1 (High-Value) above it.
  • Model Comparison — Trained and tuned Logistic Regression (80.06%), Decision Tree (83.99%), and Random Forest (87.61%) using GridSearchCV with 5-fold cross-validation.
  • Key Finding — Property area and city location were the dominant price drivers, accounting for ~70% of the Random Forest's feature importance.

Results

Random Forest outperformed both baseline models. Its ensemble of 50 decision trees (max depth = 10) reduced variance and handled the non-linear relationship between city location and price that Logistic Regression couldn't capture.
