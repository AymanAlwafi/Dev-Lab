Online Notifier

A simple, lightweight Discord bot that instantly notifies your server when someone joins a voice channel.

Python Discord API discord.py Open source

About

Online Notifier is a simple, lightweight Discord bot that instantly notifies your server when someone joins a voice channel. It helps keep everyone informed and improves engagement in gaming, study, or chill servers by making it easier to see who's active.

Many Discord servers go quiet simply because members don't know who's online. That was a problem I faced in my own server — and that's where this bot idea came to mind. Online Notifier alerts everyone when someone joins a voice channel, bringing attention back to activity and encouraging conversations and participation.

Features

  • Sends a notification message when someone joins a voice channel.
  • To exclude certain channels, simply hide them from the bot — it won't see them and won't send notifications for them.
  • Lightweight and fast with minimal resource usage.

Setup

After inviting the bot to your server, set the notification channel by running the following command inside the channel you want notifications in:

!setchannel

The bot will immediately start sending notifications to that channel whenever someone joins a voice channel.
